Let's Get Started

On this page, you can download HPIpy package, and find additional resources to help you get started.


Download

Obtain the latest version of HPIpy using one of the options below:

  1. Using link: wget https://kaabil.net/hpipy/downloads/hpipy.tar.gz

  2. Clone GitHub repository: git clone https://github.com/usubioinfo/hpipy.git

Installation Instructions

For detailed installation instructions, please refer to the instructions on the User Guide page.

Dependencies

  • For Gene Ontology (GO) Similarity model, InterProScan is required to be installed locally, which can be obtained from InterPro.
  • Alternatively, the users can provide 2 files (one for host and other for pathogen species) containing protein IDs and their respective GO term(s) in a comma- or tab-separated file
